Sichtbarkeitsmodifier

  • public - Bestimmt ob die Klasse oder Methode für Programmcode anderer Klassen sichtbar ist
    also jeder diese Methode verwenden kann.

  • privat - Hier kann nur das Objekt selbst diese Methode nutzen. Sinnvoll wenn
    Methoden benutzt werden um die Komplexität zu verkleinern und Probleme zu lösen.

  • protected - Nur abgeleitete Klassen oder Klassen im gleichen Verzeichnis(Paket) können diese Methode nutzen

  • static - Erlaubt das die methode aufgerufen werden kann ohne ein Objekt oder Klasse zu bilden
    Es kann also nur mit dem Klassennamen aufgerufen werden. Fehlt das static so muss zwingend ein Objekt
    erzeugt werden.

  • void - heisst auf deutsch 'frei'. => Die Methode liefert keinen Rückgabewert.

  • final - Zur definition von Variablen die nicht geändert werden können sollen. z.B Konstanten
    Bei einer Vererbung ist es der Unterklasse nicht erlaubt den Programmcode der Methode zu ändern.
